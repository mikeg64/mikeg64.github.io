Detailed Description

Functions to handle static mesh refinement (SMR).

PURPOSE: Functions to handle static mesh refinement (SMR).

REFERENCES:

  • G. Toth and P.L. Roe, "Divergence and Curl-preserving prolongation and restriction formulas", JCP 180, 736 (2002)

CONTAINS PUBLIC FUNCTIONS:

  • RestrictCorrect(): restricts (averages) fine Grid solution to coarse, and corrects cells at fine/coarse boundaries using restricted fine Grid fluxes
  • Prolongate(): sets BC on fine Grid by prolongation (interpolation) of coarse Grid solution into fine grid ghost zones
  • SMR_init(): allocates memory for send/receive buffers

PRIVATE FUNCTION PROTOTYPES:

  • ProCon() - prolongates conserved variables
  • ProFld() - prolongates face-centered B field using TR formulas
  • mcd_slope() - returns monotonized central-difference slope
