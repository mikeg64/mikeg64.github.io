\doxysubsection{Detailed Description}
Writes Portable Gray Map (P\+GM) outputs. 

P\+U\+R\+P\+O\+SE\+: Writes Portable Gray Map (P\+GM) outputs. These are extremely simple grayscale 2D images, see e.\+g. sourceforge for documentation. With S\+MR, dumps are made for all levels and domains, unless nlevel and ndomain are specified in $<$output$>$ block.
