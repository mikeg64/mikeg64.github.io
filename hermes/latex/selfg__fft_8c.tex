\doxysubsection{Detailed Description}
Contains functions to solve Poisson\textquotesingle{}s equation for self-\/gravity in 1D, 2D and 3D using F\+F\+Ts (actually, the 1D algorithm uses Forward Elimination followed by Back Substitution\+: F\+E\+BS). 

These functions require P\+E\+R\+I\+O\+D\+IC B\+Cs and use the Jeans swindle.

The 2D and 3D f\textquotesingle{}ns use F\+F\+T\+W3.\+x, and for M\+PI parallel use Steve Plimpton\textquotesingle{}s block decomposition routines added by N. Lemaster to /athena/fftsrc. This means to use these fns the code must be
\begin{DoxyItemize}
\item (1) configured with --with-\/gravity=fft --enable-\/fft
\item (2) compiled with links to F\+F\+TW libraries (may need to edit Makeoptions)
\end{DoxyItemize}

For N\+O\+N-\/\+P\+E\+R\+I\+O\+D\+IC B\+Cs, use selfg\+\_\+multig() functions.
