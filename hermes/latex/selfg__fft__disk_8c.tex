\doxysubsection{Detailed Description}
Contains functions to solve Poisson\textquotesingle{}s equation for self-\/gravity in disk symmetry, in 1D, 2D and 3D using F\+F\+Ts. 

For 1D, x1 is perpendicular to the plane For 2D, x1 is in plane and periodic, and x2 is perpendicular to the plane For 3D, x1 and x2 are in plane and periodic, and x3 is perpendicular to the plane

The F\+FT\textquotesingle{}s use the F\+F\+T\+W3.\+x libraries, and for M\+PI parallel use Steve Plimpton\textquotesingle{}s block decomposition routines added by N. Lemaster to /athena/fftsrc. This means to use these fns the code must be (1) configured with --enable-\/fft (2) compiled with links to F\+F\+TW libraries

For N\+O\+N-\/\+P\+E\+R\+I\+O\+D\+IC B\+Cs, use selfg\+\_\+multig() functions. For F\+U\+L\+L\+Y-\/\+P\+E\+R\+I\+O\+D\+IC B\+Cs, use selfg\+\_\+fft functions

N\+O\+TE\+: The functions in selfg\+\_\+fft assume P\+E\+R\+I\+O\+D\+IC BC in A\+LL directions. The functions here implement O\+P\+EN BC in O\+NE direction and P\+E\+R\+I\+O\+D\+IC BC in the other direction(s).
