\doxysubsection{Detailed Description}
Computes timestep using C\+FL condition. 

P\+U\+R\+P\+O\+SE\+: Computes timestep using C\+FL condition on cell-\/centered velocities and sound speed, and Alfven speed from face-\/centered B, across all Grids being updated on this processor. With M\+PI parallel jobs, also finds minimum dt across all processors.

For special relativity, the time step limit is just (1/dx), since the fastest wave speed is never larger than c=1.

A C\+FL condition is also applied using particle velocities if P\+A\+R\+T\+I\+C\+L\+ES is defined.
