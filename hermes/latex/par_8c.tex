\doxysubsection{Detailed Description}
Set of routines to provide simple access to a R/O parameter file. 

P\+U\+R\+P\+O\+SE\+: set of routines to provide simple access to a R/O parameter file slightly modeled after F\+O\+R\+T\+R\+AN namelist. Parameters can also be read directly from the commandline in the same format.

E\+X\+A\+M\+P\+LE of input file in \textquotesingle{}par\textquotesingle{} format\+:
\begin{DoxyItemize}
\item $<$blockname1$>$ \# block name; should be on a line by itself
\item name1 = value1 \# whitespace around the = is optional
\item \# blank lines between blocks are OK
\item $<$blockname2$>$ \# start new block
\item name1 = value1 \# note that name1 can appear in different blocks
\item name2 = value2 \#
\item 
\item $<$blockname1$>$ \# same blockname can re-\/appear, though is a bit odd
\item name3 = value3 \# this would be the 3rd name in this block
\end{DoxyItemize}

L\+I\+M\+I\+T\+A\+T\+I\+O\+NS\+:
\begin{DoxyItemize}
\item M\+A\+X\+L\+EN means static strings; could make fancier dynamic string reader
\item blocknames and parameters (key=val\+::comment) are all single-\/line based
\end{DoxyItemize}
