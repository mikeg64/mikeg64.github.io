\doxysubsection{Detailed Description}
Problem generator for firehose test of Braginskii viscosity. 

P\+U\+R\+P\+O\+SE\+: Problem generator for firehose test of Braginskii viscosity.
\begin{DoxyItemize}
\item iprob=1\+: vortex test suggested by Steve Cowley, developed by Greg Hammett
\item iprob=2\+: shear test from Appendix C.\+3.\+4 in Prateek Sharma\textquotesingle{}s thesis.
\end{DoxyItemize}

P\+R\+I\+V\+A\+TE F\+U\+N\+C\+T\+I\+ON P\+R\+O\+T\+O\+T\+Y\+P\+ES\+:
\begin{DoxyItemize}
\item ran2() -\/ random number generator from NR
\item pbc\+\_\+ix1() -\/ sets B\+Cs on L-\/x1 (left edge) of grid used in shear test
\item pbc\+\_\+ox1() -\/ sets B\+Cs on R-\/x1 (right edge) of grid used in shear test 
\end{DoxyItemize}
