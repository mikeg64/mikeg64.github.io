\doxysubsection{Detailed Description}
Implements various optically thin cooling functions. ~\newline
 

These can be enrolled by setting Cooling\+Func=N\+A\+ME in the problem generator, where N\+A\+ME is one of the functions in this file.

Each cooling function returns the cooling rate per volume. The total (or equivalently the internal) energy then evolves as
\begin{DoxyItemize}
\item d\+E/dt = de/dt = -\/ Cooling\+Func
\end{DoxyItemize}

Some of these cooling functions return the cooling rate per volume in cgs units \mbox{[}ergs/cm$^\wedge$\{3\}/s\mbox{]}. Thus, to use these functions, the entire calculation must be in cgs, or else the cooling rate has to scaled appropriately in the calling function.

To add a new cooling function, implement it below and add the name to \mbox{\hyperlink{microphysics_2prototypes_8h}{src/microphysics/prototypes.\+h}}. Note the argument list must be (d,P,dt).
