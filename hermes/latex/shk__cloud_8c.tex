\doxysubsection{Detailed Description}
Problem generator for shock-\/cloud problem; planar shock impacting a single spherical cloud. 

P\+U\+R\+P\+O\+SE\+: Problem generator for shock-\/cloud problem; planar shock impacting a single spherical cloud. Input parameters are\+:
\begin{DoxyItemize}
\item problem/\+Mach = Mach number of incident shock
\item problem/drat = density ratio of cloud to ambient
\item problem/beta = ratio of Pgas/\+Pmag
\item problem/iprob = integer flag to determine problem
\end{DoxyItemize}

The cloud radius is fixed at 1.\+0. The center of the coordinate system defines the center of the cloud, and should be in the middle of the cloud. The shock is initially at x1=-\/2.\+0. A typical grid domain should span x1 in \mbox{[}-\/3.\+0,7.\+0\mbox{]} , y and z in \href{see input file in /tst}{\texttt{ -\/2.\+5,2.\+5}} Various test cases are possible\+:
\begin{DoxyItemize}
\item (iprob=1)\+: B parallel to shock normal
\item (iprob=2)\+: B perpendicular to shock normal -- N\+OT Y\+ET I\+M\+P\+L\+E\+M\+E\+N\+T\+ED
\end{DoxyItemize}

If the code is configured with nscalars$>$0, the cloud material is labeled with U\mbox{[}k\mbox{]}\mbox{[}j\mbox{]}\mbox{[}i\mbox{]}.s\mbox{[}0\mbox{]}=1. ~\newline
