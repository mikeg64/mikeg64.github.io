\doxysubsection{Detailed Description}
Problem generator for steady planar gravitational flow in a simple 1D gravitational field. 

P\+U\+R\+P\+O\+SE\+: Problem generator for steady planar gravitational flow in a simple 1D gravitational field\+: g = grav$\ast$cos(k\+\_\+par$\ast$x) with periodic boundary conditions. The 1D flow can be initialized in a 3D (x1,x2,x3) domain using the following transformation rules\+:
\begin{DoxyItemize}
\item x = x1$\ast$cos(alpha) + x2$\ast$sin(alpha)
\item y = -\/x1$\ast$sin(alpha) + x2$\ast$cos(alpha)
\item z = x3
\end{DoxyItemize}

This problem is a good test of the source terms in a static grav potential.
