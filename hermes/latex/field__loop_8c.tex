\doxysubsection{Detailed Description}
Problem generator for advection of a field loop test. 

P\+U\+R\+P\+O\+SE\+: Problem generator for advection of a field loop test. Can only be run in 2D or 3D. Input parameters are\+:
\begin{DoxyItemize}
\item problem/rad = radius of field loop
\item problem/amp = amplitude of vector potential (and therefore B)
\item problem/vflow = flow velocity
\item problem/drat = density ratio in loop. Enables density advection and thermal conduction tests. The flow is automatically set to run along the diagonal.
\end{DoxyItemize}

Various test cases are possible\+:
\begin{DoxyItemize}
\item (iprob=1)\+: field loop in x1-\/x2 plane (cylinder in 3D)
\item (iprob=2)\+: field loop in x2-\/x3 plane (cylinder in 3D)
\item (iprob=3)\+: field loop in x3-\/x1 plane (cylinder in 3D)
\item (iprob=4)\+: rotated cylindrical field loop in 3D.
\item (iprob=5)\+: spherical field loop in rotated plane
\end{DoxyItemize}

A sphere of passive scalar can be added to test advection of scalars.

The temperature in the loop can be changed using drat to test conduction.
