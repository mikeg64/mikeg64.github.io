\doxysubsection{Detailed Description}
Computes 1D fluxes using the Harten-\/\+Lax-\/van Leer (H\+L\+LE) Riemann solver. 

P\+U\+R\+P\+O\+SE\+: Computes 1D fluxes using the Harten-\/\+Lax-\/van Leer (H\+L\+LE) Riemann solver. This flux is very diffusive, especially for contacts, and so it is not recommended for use in applications. However, as shown by Einfeldt et al.(1991), it is positively conservative, so it is a useful option when other approximate solvers fail and/or when extra dissipation is needed.
