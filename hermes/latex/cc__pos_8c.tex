Functions to compute (x1,x2,x3) positions of cells i,j,k. ~\newline
 

P\+U\+R\+P\+O\+SE\+: Functions to compute (x1,x2,x3) positions of cells i,j,k. ~\newline
 In a nested grid, each Grid structure is a patch in a larger computational domain (with the exception of the level0 grid). The displacement of the origin of the Grid from the origin of the computational domain (level0 grid) is x1\+\_\+\{disp\} = idisp$\ast$dx1. Furthermore, the origin of the level0 grid can be displaced by a distance x1\+\_\+\{0\} from the origin of x1. Thus, the x1 position of the center of cell i (x1\+\_\+\{cc,i\}) in any level Grid is x1\+\_\+\{cc,i\} = x1\+\_\+\{0\} + ((i + idisp) + 0.\+5)$\ast$dx1 Similarly for x2 and x3.
