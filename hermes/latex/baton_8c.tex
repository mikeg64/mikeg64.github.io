\doxysubsection{Detailed Description}
Stage independent processes by passing a baton. 

P\+U\+R\+P\+O\+SE\+: Stage independent processes by passing a baton. The functions here assume that no M\+PI communication will proceed between \mbox{\hyperlink{prototypes_8h_ab012dc3459875830b6e80a5dbf9ba110}{baton\+\_\+start()}} and \mbox{\hyperlink{prototypes_8h_a206f64b6c68459e4ec7789ab92b65665}{baton\+\_\+stop()}} and that the arguments to the functions will be identical. This is easy enough to accomplish, but can be error prone. Use these functions with caution!

These functions were written and added to athena by T. A. Gardiner in March, 2008.
