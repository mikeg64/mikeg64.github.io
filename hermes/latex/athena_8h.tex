\doxysubsection{Detailed Description}
Contains definitions of many data types and structures. 

P\+U\+R\+P\+O\+SE\+: Contains definitions of the following data types and structures\+:
\begin{DoxyItemize}
\item Real -\/ either float or double, depending on configure option
\item \mbox{\hyperlink{struct_cons_s}{ConsS}} -\/ cell-\/centered conserved variables
\item \mbox{\hyperlink{struct_prim_s}{PrimS}} -\/ cell-\/centered primitive variables
\item \mbox{\hyperlink{struct_cons1_d_s}{Cons1\+DS}} -\/ conserved variables in 1D\+: same as \mbox{\hyperlink{struct_cons_s}{ConsS}} minus Bx
\item \mbox{\hyperlink{struct_prim1_d_s}{Prim1\+DS}} -\/ primitive variables in 1D\+: same as \mbox{\hyperlink{struct_prim_s}{PrimS}} minus Bx
\item \mbox{\hyperlink{struct_grain_s}{GrainS}} -\/ basic properties of particles
\item \mbox{\hyperlink{struct_grid_s}{GridS}} -\/ everything in a single Grid
\item \mbox{\hyperlink{struct_domain_s}{DomainS}} -\/ everything in a single Domain (potentially many Grids)
\item \mbox{\hyperlink{struct_mesh_s}{MeshS}} -\/ everything across whole Mesh (potentially many Domains)
\item \mbox{\hyperlink{struct_output_s}{OutputS}} -\/ everything associated with an individual output ~\newline
 
\end{DoxyItemize}
