\doxysubsection{Detailed Description}
Contains functions to solve Poisson\textquotesingle{}s equation for self-\/gravity in 3D using F\+F\+Ts, using O\+P\+EN B\+Cs in all three directions. 

The function uses F\+F\+T\+W3.\+x, and for M\+PI parallel use Steve Plimpton\textquotesingle{}s block decomposition routines added by N. Lemaster to /athena/fftsrc. This means to use these fns the code must be
\begin{DoxyItemize}
\item (1) configured with --with-\/gravity=fft\+\_\+obc --enable-\/fft
\item (2) compiled with links to F\+F\+TW libraries (may need to edit Makeoptions)
\end{DoxyItemize}

For P\+E\+R\+I\+O\+D\+IC B\+Cs, use selfg\+\_\+fft() functions.
