\doxysubsection{Detailed Description}
Functions to convert conservative-\/$>$primitive variables, and $<$-\/$>$ 

P\+U\+R\+P\+O\+SE\+: Functions to convert conservative-\/$>$primitive variables, and vice-\/versa. Routines for both Newtonian and special relativistic physics are included. Also contains function to compute fast magnetosonic speed for Newtonian flows only.

C\+O\+N\+T\+A\+I\+NS P\+U\+B\+L\+IC F\+U\+N\+C\+T\+I\+O\+NS\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{convert__var_8c_ae7dfb4da1bb433bf329ddd2a7a6fefb0}{Cons\+\_\+to\+\_\+\+Prim()}} -\/ converts Cons type to Prim type
\item \mbox{\hyperlink{convert__var_8c_a168e092d771134a92ff8be7e624d51d9}{Cons1\+D\+\_\+to\+\_\+\+Prim1\+D()}} -\/ converts 1D vector (Bx passed through arguments)
\item \mbox{\hyperlink{convert__var_8c_a030b3be2910e01b12ca3a425196aa77b}{Prim1\+D\+\_\+to\+\_\+\+Cons1\+D()}} -\/ converts 1D vector (Bx passed through arguments)
\item \mbox{\hyperlink{convert__var_8c_a014d3ca3dda378d010d612b454eccd93}{cfast()}} -\/ computes fast magnetosonic speed
\end{DoxyItemize}

For special relativity, there are two versions of the Cons1\+D\+\_\+to\+\_\+\+Prim1D functions, one for H\+Y\+D\+RO and one for M\+HD. There is also a \textquotesingle{}check\+\_\+\+Prim\textquotesingle{} and \textquotesingle{}check\+\_\+\+Prim1D\textquotesingle{} routine for S\+R\+M\+HD which returns a primitve state without verifying physical status.
