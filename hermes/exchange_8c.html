Detailed Description

Exchange the gas-particle coupling array between ghost and boundary zones.

PURPOSE: Particles near grid boundaries deposit their physical properties partially to the ghost zones. This part of the deposit is to be mapped to the grid zone. The procedure is opposite to setting boundary conditions.
