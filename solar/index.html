Windows

We will be using nano during the shell session. In order to install nano for GitBash on Windows:
1. Install GitBash and open a terminal window.
2. Enter the command (copy and paste it into GitBash window - to paste in GitBash right click on the top of the terminal window and select Editand then Paste): curl -O https://raw2.github.com/apawlik/2014-01-14-manchester/gh-pages/setup/install_nano_GitBash.sh
3. Enter the command: ./install_nano_GitBash.sh
4. Type in nano and hit enter, you should see nano opened in your terminal window.

MATLAB

The attendees will need access to MATLAB R2013b and Statistics Toolbox. Trial licenses for all products will be available for the duration of the training.

As long as the attendees have a valid academic email address they should be able to access the online tutorials. There is a lot of material available, so even if you have some previous experience of MATLAB, it should be possible to find something useful there prior to the course.
